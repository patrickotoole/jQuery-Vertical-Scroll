UI Scroll Demo

About

Welcome to the World's ugliest newspaper demo for jQuery UI scroll bars with nesting and inline styling. There are 5 demonstrations occurring on the following classes:
  • Outer - shows the basic scroll bar applied to a div of fixed height. (this header is included within the outer)
  • Demo 1 - shows scroll bar styled using inline properties and jquery after completion event is trigger.
  • Demo 2 - shows scroll bar styled using inline properties and jquery without using completion event
  • Demo 3 - shows styling without applying inline style to the wrapper
  • Demo 4 - shows case of inline style (with padding) applied to both the interior and wrapper

Outer - Basic

Outer scrollbar is initialized first. Inner scrollbar initialization occurs after to preserve jQuery UI functionality. jQuery UI objects cannot be cloned (easily). Since the scrollbar is based on them, must proceed from the outer most div and move inward when applying scrollbars.

Demo 1 - Styling Options with Event Trigger

Four techniques associated with the scrollbar are demonstrated in this demo:

  1. Scrollbar options {end_cap_size : 10}
  2. Using completion trigger to style
  3. Removing scrollbars
  4. Creating scrollbars

When scrollbar is remove, options are saved. However, dynamic styling is lost unless styling occurs during completion trigger.

remove scrollbar

Without event trigger, this custom styling could occur with the exception of removing the html from the arrows. All other styling issue could be handled externally in a style sheet.

Note: both the wrapper and content inherit from the inline style. Demo 3 shows how this could be disabled.

Demo 2 - Styling Options without Event Trigger

Similar to Demo 1 but styling of scrollbars not related to event

  1. Scrollbar options {width:36,incrementSize:2,end_cap_size:20}
  2. Restyled slider (radius, color)
  3. Added border to content area
  4. Inline margin applied to both content and outer

When scrollbar is removed, options are saved but dynamic styling is not reapplied when scrollbar is redrawn.

remove scrollbar

Demo 3 - Content vs. Wrapper

inlineStyleForWrapper = false
Scrollbar options {end_cap_size : 0, inlineStyleForWrapper:false, location:right}

None of the inline style is applied to the wrapper, making the border shown in Demo 2 easier to implement. A list follows describing all that is occurring in this demo.

  • padding/bgcolor on interior but not exterior wrapper
  • added display:inline-block after drawn
  • no vertical align added but was on inline style
  • scrollbar background radius
  • width
